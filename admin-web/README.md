# Heracles Admin — веб-морда (один файл)

`index.html` — самодостаточная админка на чистом JS, без сборки. Логин → настройка
провайдера ИИ, ключей, моделей и промптов → управление админами. Всё через браузер.

Ключи провайдеров никогда не показываются (только статус «ключ задан»); все запросы
идут с `Authorization: Bearer`.

## Как пользоваться

**Вариант 1 — локально.** Просто откройте `index.html` двойным кликом. В поле
«Backend URL» укажите адрес сервера (по умолчанию `https://heracles-admin-server.onrender.com`).

**Вариант 2 — задеплоить как Render Static Site:**

- New → Static Site → этот репозиторий.
- **Root Directory:** `admin-web`
- **Build Command:** оставить пустым
- **Publish Directory:** `.`

После деплоя добавьте домен этой статики в `CORS_ORIGINS` бэкенда и сделайте redeploy.

## Важно про CORS

Бэкенд отдаёт ответы только тем источникам (origin), что перечислены в `CORS_ORIGINS`.

- Если открываете файл локально (`file://`) — origin будет `null`. Чтобы работало,
  либо **временно** уберите/очистите `CORS_ORIGINS` на бэкенде (тогда разрешены все
  origins), либо запустите файл через локальный сервер и добавьте его origin в список.
- Если деплоите как Static Site — добавьте её домен (`https://<name>.onrender.com`) в
  `CORS_ORIGINS`.

Настройки (Backend URL, токен) хранятся в `localStorage` браузера. Выход стирает токен.
