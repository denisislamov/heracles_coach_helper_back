# Картинки лендинга

Положи сюда два файла (те, что присылал в чат):

- `mascot.png` — квадратный робот с табло «КБЖУ». Используется как логотип,
  иконка сайта (favicon) и иллюстрация в hero-блоке.
- `hero.png` — широкая инфографика «Жиромер — твой ИИ-нутрициолог». Используется
  как картинка для превью при шеринге в соцсетях/мессенджерах (og:image).

Имена файлов важны — шаблоны ссылаются именно на `mascot.png` и `hero.png`.
Формат png или jpg; если jpg — поменяй расширения в `templates/base.html`
и `templates/index.html`.
